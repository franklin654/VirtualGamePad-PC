Preferences::eventFilter The event filter virtual function is redefined to to filter for mouse and keyboard inputs when user tries to change the button-key maps. Checks which object is sending the event and type of event. If event is a keyboard or mousebutton press than map and the object is buttonmap than get the virtual key code of the key pressed and store the change in a temporary variable.
